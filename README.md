# Tic-Tac-Toe
Tic Tac Toe ou Jogo da Velha criado com HTML, CSS e Javascript com JQuery

- 2 Jogadores simultâneamente
- Armazena scores
- Possibilita surrender
- Possibilita troca de estilo de layout
